# Notebooks

Demo notebooks covering the backcast engine end-to-end on the Tier 2
synthetic dataset generated by the sibling `synthetic_data_generator` project.

| Notebook | Topic |
|---|---|
| `01_eda.ipynb` | Dataset structure, missingness, correlations, rolling vol |
| `02_model_comparison.ipynb` | EM vs Kalman TVP vs regime HMM |
| `03_validation.ipynb` | Walk-forward holdout, PIT histogram, residuals |
| `04_downstream.ipynb` | Covariance, uncertainty sets, backtests, copula |

## Running

```bash
# build + execute all four
python scripts/build_notebooks.py

# just build, don't execute
python scripts/build_notebooks.py --no-run
```

The notebooks add `backcast_engine/src` to `sys.path` at the top so
they don't require an editable install of the package.
